Bright Tech Education Responsive Portfolio Website
A complete responsive landing page designed for Bright Tech Education,
Showcasing ICT, Coding/Robotics, and Graphic Design training programs.
This project highlights our mission to equip learners with 21st century
digital skills that foster creativity, problem-solving, and lifelong learning.

Features

- Responsive layout for desktop, tablet, and mobile
- Horizontal navigation bar (Home, About, Contact, Portfolio)
- Off-white quality background covering the entire page
- Bright Tech logo displayed on the left side
- Hamburger menu for mobile navigation
- Smooth scrolling and fade-in animation effects
- instructor section with profile and bio
- Click-to-call phone numbers and Whatsapp links
- Social media icons using offline Font Awesome
- Scroll-to-Top button
- Copyright footer

Project Structure

Applications Used

* HTML5 - Structure of the website
* CSS3 - Styling, responsiveness, and animations
* JavaScript (JS) - Interactivity (toggle menu, fade effects, scroll to top)
* Font Awesome Offline - Icons for contact and social media
